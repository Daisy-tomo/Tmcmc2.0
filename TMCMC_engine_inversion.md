## 2. 问题背景

已知发动机整机性能观测量：

- **比推力** $R_{ud}$ （N·s/kg）
- **比油耗** $C_{ud}$ （kg/(N·h)）

希望反推出 4 个经敏感性筛选后的关键修正参数的**后验概率分布**：

| 参数符号 | 物理含义 | 先验区间 | 真值（用于生成合成数据） |
|----------|----------|----------|--------------------------|
| $\eta_k$ | 压气机绝热效率 | [0.840, 0.860] | 0.850 |
| $\eta_t$ | 涡轮绝热效率 | [0.860, 0.920] | 0.890 |
| $\eta_T$ | 燃烧放热系数 | [0.970, 0.990] | 0.980 |
| $\eta_m$ | 机械效率 | [0.980, 0.995] | 0.988 |

其余 9 个不敏感参数固定在标称值，不参与反演。

## 3. 算法：TMCMC（过渡 Markov 链蒙特卡罗）

### 3.1 基本思想

TMCMC 由 Ching & Beck（2007）提出，核心思路是将困难的后验采样问题
分解为一系列**逐步"升温"的中间目标分布**：

$$p_j(\theta) \propto p(\theta)\, [p(\mathbf{d}|\theta)]^{\beta_j}, \quad 0 = \beta_0 < \beta_1 < \cdots < \beta_J = 1$$

- $\beta_j = 0$：先验分布（容易采样）
- $\beta_j = 1$：完整后验分布（目标）

每一阶段（Stage）通过**重要性重采样 + 短 MH 链**从 $p_j$ 过渡到 $p_{j+1}$。

### 3.2 算法流程（每一阶段）

```
Stage j：当前粒子集 {θ_i} 服从 p_j

1. 自适应确定 β_{j+1}
   目标：权重 w_i = L(θ_i)^{Δβ} 的变异系数 COV ≈ 1.0
   方法：二分法搜索 Δβ

2. 计算归一化重要性权重
   W_i = w_i / Σ w_i

3. 计算加权样本协方差 C_w（用于 MH 提议）

4. 系统重采样（低方差重采样器）
   从 {θ_i} 按权重 W_i 抽取 N 个粒子

5. MH 扰动（短链）
   提议分布：q(θ'|θ) = N(θ, s² · C_w)
   目标分布：p(θ)^{β_{j+1}}
   对每个粒子运行 n_mcmc 步 MH

6. 更新粒子集 → 下一阶段
```

重复至 $\beta_J = 1$，最终粒子集即后验样本。

### 3.3 关键参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `N` | 1000 | 每阶段粒子数；增大可提高精度，但计算量线性增长 |
| `cov_tgt` | 1.0 | 权重 COV 目标值；标准取值，控制每次 β 步长 |
| `n_mcmc` | 4 | 每粒子每阶段 MH 步数；3-5 通常足够 |
| `s2` | 0.04 | 提议协方差全局缩放因子 $(0.2)^2$ |

> **运行轻量化设计**：N=1000、n_mcmc=4，在普通笔记本（无 GPU）上通常
> 可在 1-5 分钟内完成（具体取决于 stage 数量，一般 10-20 个 stage）。

## 4. 前向物理模型

发动机热力循环正向计算流程（`engine_fwd` 函数）：

```
飞行条件 → 进口冲压比 τ_v, 进口总温 T_B
         → 压气机出口温度 T_k
         → 相对耗油量 g_T
         → 热恢复系数 λ_heat
         → 各段总压恢复 σ_bx
         → 单位自由能 L_sv
         → 最优分配系数 x_pc
         → 喷管出口速度 V_j1, V_j2
         → 比推力 R_ud  /  比油耗 C_ud
```

分段热力学函数：

- $k_T$（燃气绝热指数）：$T_g > 1600$ K → 1.25；> 1400 K → 1.30；其他 → 1.33
- $R_T$（燃气气体常数）：分段对应 288.6 / 288.0 / 287.6 J/(kg·K)
- $\delta$（涡轮冷却引气系数）：$\delta = \text{clip}[0.02 + (T_g-1200)/100 \times 0.02,\ 0,\ 0.15]$

---

## 5. 似然函数

采用高斯噪声假设，观测噪声比例为 0.1%：

$$\ln p(\mathbf{d}|\theta) = -\frac{1}{2}\left[\left(\frac{R_{obs} - R_{pred}}{\sigma_R}\right)^2 + \left(\frac{C_{obs} - C_{pred}}{\sigma_C}\right)^2\right]$$

> 常数项 $-\frac{1}{2}\ln(2\pi\sigma^2)$ 在 MH 比值中消去，故略去。

## 6. 程序结构

```
TMCMC_engine_inversion.m
│
├── TMCMC_engine_inversion()          ← 主函数（入口）
│   ├── 参数定义与先验边界
│   ├── 固定参数与运行条件
│   ├── 生成合成观测数据
│   ├── 调用 tmcmc_core()
│   └── 输出统计结果 + 调用绘图
│
├── tmcmc_core()                      ← TMCMC 主循环
│   ├── Stage 0: 先验均匀采样
│   └── while beta < 1:
│       ├── find_next_beta()          ← 二分法确定 β_{j+1}
│       ├── 计算权重与加权协方差
│       ├── systematic_resample()     ← 系统重采样
│       └── MH 扰动（短链）
│
├── find_next_beta()                  ← 二分法搜索合适的 β 步长
├── systematic_resample()             ← 低方差系统重采样器
├── reflect_bounds()                  ← 越界反射处理
├── loglik()                          ← 对数似然
├── engine_fwd()                      ← 发动机热力循环前向模型
├── piecewise_kT()                    ← 燃气绝热指数分段函数
├── piecewise_RT()                    ← 燃气气体常数分段函数
├── delta_cooling()                   ← 涡轮冷却引气系数
└── plot_posterior()                  ← 绘制后验分布图
```

## 8. 使用方法

1. 将 `TMCMC_engine_inversion.m` 复制到 MATLAB 工作路径
2. 在命令窗口输入：

```matlab
TMCMC_engine_inversion
```

3. 等待运行完成（约 1-5 分钟），查看命令行输出和弹出图形

> **注意**：程序不固定随机数种子，每次运行结果会有微小随机差异，这是
> 贝叶斯蒙特卡罗方法的正常特性。

## 9. 参数调整建议

| 需求 | 调整方式 |
|------|----------|
| 提高精度 | 增大 `N`（如 2000-5000）、增大 `n_mcmc`（如 5-10） |
| 加快速度 | 减小 `N`（如 500），但结果方差增大 |
| 采样器不稳定 | 调小 `s2`（如 0.01），或检查先验区间是否合理 |
| 修改噪声水平 | 调整 `noise_frac`（第 4 节，默认 0.001） |
| 修改真值 | 调整 `theta_true` 向量 |
| 修改固定参数 | 修改 `fp` 结构体字段 |
| 修改运行条件 | 修改 `cond` 结构体字段 |
